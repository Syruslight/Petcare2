El proyecto esta detallado en las siguientes carpetas:
BD -> Scripts de las bd conforme se vayan actualizado segun requerimientos
controlador --> Clases php donde se realizan consultas para hacer las funciones requeridas
css --> Clases de estilo para el lado del front end
imagenes --> Fotos que se cargan por defecto en la pagina de la veterinaria
js --> archivo(s).js para validaciones
llamadas --> Aqui estara la interaccion entre controlador y la vista de la pagina
index.php --> Pagina que se muestra por defecto al ejecutar el proyecto (en este caso es el landing page)
